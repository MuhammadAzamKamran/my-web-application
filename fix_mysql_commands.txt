=== MySQL Troubleshooting Commands ===

Copy and paste these commands one by one into PowerShell (run as Administrator):

1. Stop MySQL processes:
Get-Process | Where-Object {$_.ProcessName -eq "mysqld"} | Stop-Process -Force

2. Check if port 3306 is free:
netstat -ano | findstr :3306

3. Check MySQL error log (last 50 lines):
Get-Content "C:\xampp\mysql\data\mysql_error.log" -Tail 50

4. Check if MySQL is running:
Get-Process | Where-Object {$_.ProcessName -eq "mysqld"}

5. Test MySQL connection manually:
cd C:\xampp\mysql\bin
.\mysql.exe -u root

6. Check if MySQL is listening on port 3306:
netstat -ano | findstr :3306
